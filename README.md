# Qard

Aplicación de gestión de gastos desarrollada con React Native,
TypeScript y Supabase.

------------------------------------------------------------------------

## 🛠 Stack

-   React Native
-   TypeScript
-   Supabase
-   React Navigation (Stack, Drawer, Bottom Tabs)
-   Git & GitHub

------------------------------------------------------------------------

## 🚀 Instalación

``` bash
npm install
```

------------------------------------------------------------------------

## ▶️ Scripts disponibles

### Desarrollo (variables de entorno dev de Supabase)

``` bash
npm run dev
```

### Producción (variables de entorno prod de Supabase)

``` bash
npm run prod
```
